Routing Engine Cheatsheet

Standalone Pattern Matching & Dispatching Engine

URI Pattern Syntax

Named Parameter Standard
/user/{id}

Matches non-slash chars. Captures as "id".

Regex Constraint Advanced
/user/{id:\d+}

Named capture "id" restricted to digits.

Inline Regex Raw
/post/(\d+)

Raw capture group. Accessible via index "0".

Wildcard / Catch-all Fallback
/admin/*

Matches everything after the prefix.

Middleware Pipeline

01

Global

router.use('AuthMiddleware')

02

Path-Specific

router.register_path_middleware('/api/*', [...])

03

Route-Specific

router.add_route(..., middleware=['log'])

Supported Formats

  • FQN String 'app.mid.Auth'
  • Method Reference 'Auth@check'
  • Callable lambda c, r, n: ...

Quick Start Implementation

# 1. Initialize
router = Router(request)

# 2. Add Route with DI Overrides
router.add_route(
    uri='/users/{id:\d+}',
    method='GET',
    handler='app.controllers.UserController@show',
    middleware=['AuthMiddleware'],
    override_params={'timeout': 60}
)

# 3. Global Middleware
router.use(lambda c, r, next: next())

# 4. Dispatch
response = router.dispatch(container)
